Projects

CS 184: Computer Graphics & Imaging: Rasterizester

Simple rasterization, including features like supersampling, hierarchical transforms, and texture mapping with antialiasing. A functional vector graphics renderer that can take in modified SVG (Scalable Vector Graphics) files, which are widely used across the Internet.
Check out my write-up!


CS 184: Computer Graphics & Imaging: GeoMenagerie

Basic tessellation of Bezier patches, manipulation of half-edge meshes, loop subdivision (upsampling) of meshes, and shaders. A tool that loads and allows a user to edit basic COLLADA mesh files used in common modeling packages and real time graphics engines.
Check out my write-up!


CS 184: Computer Graphics & Imaging: PathTracer

Physically-based renderer using ray tracing, ray-scene intersection, acceleration structures, and physically-based lighting and materials (glass, mirrors).
Check out my write-up!


CS 184: Computer Graphics & Imaging: Lens Simulator

Augmented with a realistic camera lens simulator and autofocus, the above PathTracer is now able to recreate photographic phenemona such as depth of field, different perspectives, and defocus blurring.
Check out my write-up!


FlickFlow: Interactive Fluid Dynamics Simulation

For our CS184 final project, my partner, Kevin Chen, and I implemented an interactive fluid dynamics simulation using OpenGL. Furthermore, we integrated a Leap Motion to support user interaction with the rendered fluid using their hands.

League of Legends Ban Helper

This Google App Engine web application uses the Riot Games API for League of Legends and is written mainly in Golang. Users can look up a group of players to see statistics and get suggestions for bans when playing against those specific players. The application is intended for use in competitive playing environments.
